## Estimation: Plugin de WordPress para venta de tickets

### Scope Definition
**Included in MVP:**
- Conexión con API externa de venta de tickets.
- Importación de eventos desde el servicio externo.
- Listado público de conciertos/eventos con filtros básicos (fecha, artista, ciudad, categoría).
- Página de detalle de evento con información relevante.
- Selección de entradas y creación de pedido.
- Envío del pedido al servicio de ticketing.
- Integración/redirección a pasarela de pago.
- Confirmación de pago y actualización del estado del pedido.
- Gestión básica de errores.
- Panel de configuración en WordPress para gestionar credenciales y parámetros.

### Assumptions
- API del proveedor es bien documentada y estable.
- Requiere uso de estándares de desarrollo de plugins de WordPress.
- Se asume un nivel de seguridad básico y cumplimiento con las buenas prácticas.
- La integración con WooCommerce no se contempla.
- Se prioriza un desarrollo funcional sobre un diseño altamente personalizado.
  
### Architecture Overview
- **Frontend:** 
  - Uso de shortcodes o bloques de Gutenberg para visualización de eventos.
- **Backend:** 
  - Lógica para comunicación con la API externa y gestión de pedidos.
- **Infrastructure:** 
  - Almacenamiento de configuraciones y logs técnicos dentro de la instalación de WordPress.
