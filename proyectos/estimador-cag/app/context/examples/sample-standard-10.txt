## Estimación: Sistema interno de gestión documental para una empresa legal

### Scope Definition
**MVP incluye:**
- Acceso autenticado con roles (admin, abogado, cliente).
- Subida y almacenamiento de documentos (PDF, Word).
- Organización de documentos por casos.
- Búsqueda por texto y metadatos.
- Control de versiones de documentos.
- Registro de actividad (logs).
- Integración con servicio externo de firma digital.
- Notificaciones por correo electrónico ante cambios en documentos.
- Interfaz de usuario web orientada a productividad.

### Assumptions
- Los documentos tienen un tamaño y número manejables.
- El servicio externo de firma digital tiene un API bien documentado.
- La búsqueda por texto no incluye capacidades de procesamiento de lenguaje natural.
- El almacenamiento seguro asume el uso de S3 pero no necesita integración de terceros para el cifrado en reposo.
- La implementación contempla roles y permisos granulares por documento.

### Architecture Overview
- **Frontend:** Desarrollo de una aplicación web interactiva con autenticación y gestión de roles.
- **Backend:** API modular para la gestión de documentos, con almacenamiento en S3 y lógica para control de versiones.
- **Infrastructure:** Uso de servicios en la nube para almacenamiento seguro y manejo de correos electrónicos (por ejemplo, AWS S3 para documentos y SES para notificaciones).
