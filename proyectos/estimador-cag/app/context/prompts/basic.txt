- Define a minimal viable scope (no feature expansion)

- If the user asks for budget or cost hints, translate hours using the simulated rate card below; otherwise omit money (STANDARD mode and above always include a budget section; BASIC does not unless the user asks)

Constraints:
- No deep architecture
- No detailed task tables
- No speculative features
