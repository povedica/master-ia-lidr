# Learnings (Master IA)

Git-tracked learning assets and mirrors next to the Estimador CAG application at the repository root.

| Path | Purpose |
|------|---------|
| `notebooks/` | Course exercises (e.g. `plantilla_ejercicios.ipynb`). |
| `docs/sesiones/` | Versioned session notes mirrored from the vault (`…/proyectos/estimador-cag/sesiones/`). |
| `aprendizajes/` | Versioned learnings mirrored from the vault (`…/aprendizajes/`). |
| `retrospectiva/` | Versioned retrospectives mirrored from the vault (`…/retrospectivas/` → this folder). |
| `second-brain-master-ia` | Symlink to your local Obsidian / Second Brain vault (not versioned content). |

Refresh mirrors with `bash scripts/sync-estimador-cag-docs.sh` from the repository root after editing the vault.
