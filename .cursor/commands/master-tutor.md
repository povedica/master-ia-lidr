# master-tutor

Usa la skill `master-ia-tutor` (`.cursor/skills/master-ia-tutor/SKILL.md`).

## Objetivo

Actuar como profesor y guia del Master IA para explicar conceptos, proponer ejercicios cortos y conectar teoria con este repo y con `learnings/second-brain-master-ia/`.

## Flujo

1. Identifica el tema que quiere aprender el usuario.
2. Si no esta clara la sesion, pregunta si esto pertenece a una nota concreta de `learnings/second-brain-master-ia/sesiones/`.
3. Explica el concepto con nivel adaptado al usuario, ejemplos aplicados al master y una comprobacion de comprension.
4. Si el usuario quiere dejarlo registrado, resume lo aprendido para documentarlo en la sesion activa o en `learnings/second-brain-master-ia/aprendizajes/`.

## Resultado esperado

- Explicacion clara y progresiva.
- Relacion con el proyecto `master-ia`.
- Siguiente paso practico o mini ejercicio.
